## How to Prepare

八股文平时也不会接触，面试官也是去网上找一些问题，或者从自己整理的面试题库中找一些问题。

每个面试官问的问题是随机的，可能有一些热门问题会重复，但所有面试官面试的问题的总体范围应该是所有的面试题。所以求职者还是要把所有的面试题都背一遍。
